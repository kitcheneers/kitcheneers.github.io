# Kitcheneers Website

Welcome to the Kitcheneers website repository!

## About

This is a basic static website for Kitcheneers, hosted on GitHub Pages.

## Structure

- `index.html` - Main website page
- `styles.css` - Website styling
- `script.js` - Interactive functionality
- `README.md` - This file

## Features

- Responsive design that works on mobile, tablet, and desktop
- Navigation menu with smooth scrolling
- Hero section with call-to-action
- About section
- Projects showcase grid
- Contact section
- Smooth animations and transitions

## Getting Started

1. Clone this repository
2. Open `index.html` in your browser to view the website locally
3. Customize the content, colors, and projects to match your needs

## Customization

### Update Your Information
- Replace "Kitcheneers" with your organization/project name
- Update the project descriptions
- Change contact email in the contact section

### Change Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #2c3e50;
    --secondary-color: #3498db;
    --accent-color: #e74c3c;
}
```

### Add More Projects
Add new project cards in the projects section of `index.html`.

## Deployment

This repository is set up as a GitHub Pages site. Your website will be automatically published at `https://kitcheneers.github.io`
